# 🗓️ TaskList - Lista de Tarefas da Semana

Projeto simples e funcional feito em **HTML, CSS e JavaScript puro** que permite ao usuário criar, marcar, excluir e gerenciar suas tarefas semanais diretamente no navegador, com **persistência de dados via `localStorage`**.

---

## 🔧 Funcionalidades

- ✅ Adicionar novas tarefas
- ❌ Excluir tarefas individualmente
- 📦 Remoção em massa de tarefas marcadas
- 💾 Salvamento automático das tarefas no `localStorage`
- 📥 Carregamento automático das tarefas ao recarregar a página
- 📛 Validação de campo vazio
- 🧼 Feedback visual de ações (mensagens de sucesso/erro)

---
## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM, eventos, `localStorage`)

---

## 📁 Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/PauloAlvino/tasklist.git
   ```
2. Abra o index.html em seu navegador.
